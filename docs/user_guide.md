# Руководство пользователя BBοomer

## Введение

BBοomer - это бомбер на Python без авто перехода на сайт и без КД на отправку. Включает детальные логи и отображение процента выполнения атаки.

## Установка

1. Убедитесь, что у вас установлен Python 3.8 или выше
2. Скачайте репозиторий BBοomer
3. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

## Запуск приложения

Для запуска приложения выполните команду:
```
python src/main.py
```

## Использование

### Основные функции

1. **Атака SMS/звонками** - Отправка множества сообщений или звонков жертве
2. **Без авто перехода на сайт** - Все действия выполняются в фоновом режиме
3. **Без КД на отправку** - Отсутствие задержек между отправками сообщений
4. **Детальные логи** - Полное логирование всех действий
5. **Отображение процента выполнения** - Визуализация прогресса атаки

### Меню приложения

1. Запустить атаку
2. Настройки атаки
3. Просмотр логов
4. Остановить атаку
5. Выход

### Запуск атаки

1. Выберите пункт "Запустить атаку" в меню
2. Введите номер телефона жертвы
3. Укажите количество сообщений для отправки (по умолчанию 100)
4. Программа автоматически начнет атаку

### Просмотр логов

1. Выберите пункт "Просмотр логов" в меню
2. Будут отображены последние 20 записей из лога
3. Также можно выполнить поиск по ключевому слову в логах

### Остановка атаки

1. В любой момент можно остановить атаку, выбрав пункт "Остановить атаку"
2. Программа завершит текущие операции и остановится

## Настройка

Конфигурация программы находится в файле `config/settings.json`:

- `target_phone` - Номер телефона цели по умолчанию
- `message_count` - Количество сообщений по умолчанию
- `services` - Список активных сервисов
- `cooldown` - Задержка между отправками (0 для отсутствия КД)
- `log_level` - Уровень логирования (basic, detailed, debug)

## Важное предупреждение

Этот инструмент предназначен только для образовательных целей и этичного тестирования. Использование для незаконных действий преследуется по закону. Убедитесь, что у вас есть разрешение на тестирование целевых систем.
